/*------------------------------------------------------------------
[GLOSSARY]

**Данные пользователя, учетные данные:
  full-name - полное имя, ФИО;
  nickname - псевдоним;
  sex - пол;
  first-name - имя;
  middle-name - отчество;
  last-name - фамилия;
  user — пользователь;
  userpic, avatar — юзерпик, маленькая картинка пользователя;
  author - автор записи или комментария;
  profile - профиль (пользователя);
  administrator, (admin) - администратор;
  moderator - модератор;
  chief, director, leader, mamager - руководитель;
  mentor, master - наставник, учитель;
  operator, (oper) - оператор;
  staff - персонал, штатный сотрудник;
  member - участник, представитель;
  manager, (mgr) - менеджер;
  redactor (red), editor (ed) - редактор;
  phone-number, phone, tel - телефон;
  e-mail, email - эл. почта;
  logout - выход;
  login - вход;

  user agreement - пользовательское соглашение;
  terms of use - условия использования;
  privacy policy - политика конфиденциальности;
  Terms of Service (ToS) - условия обслуживания;

**Блоки, компоненты:
  page — корневой элемент страницы;
  header — шапка (страницы или элемента);
  footer — подвал (страницы или элемента);
  section — раздел контента (один из нескольких);
  body — основная часть (страницы или элемента);
  content — содержимое элемента;
  sidebar — боковая колонка (страницы или элемента);
  aside — блок с дополнительной информацией;
  widget — виджет, например, в боковой колонке;
  hero - Броский заголовок страницы с фонофой картинкой;
  nav - навигация;
  module - модуль;
  socials — блок иконок соцсетей;
  slider, carousel — слайдер;
  pagination — постраничная навигация;
  breadcrumbs — навигационная цепочка, «хлебные крошки»;
  modal — модальное (диалоговое) окно;
  popup — всплывающее окно;
  tooltip, tip — всплывающие подсказки;
  preview — анонс, отрывок, например новости или поста, может состоять из заголовка, описания и картинки. Предполагается ссылка на полную версию;
  card - карточка;

**Раскладка, структурные блоки:
  wrapper, (wrap) — обёртка, обычно внешняя;
  inner — внутренняя обёртка;
  outer - внешняя обертка;
  container, holder, box — контейнер;
  grid — раскладка (страницы или элемента) в виде сетки (обычно содержит в себе row и col);
  row — контейнер в виде строки;
  col, column — контейнер в виде столбца;

**Декор:
  line - линия (разделительная);
  arrow - стрелка;
  divider - декоративный элемент короткая полоска;

**Элементы управления:
  button, (btn) — кнопка;
  control — элемент управления, например, стрелки «Вперёд/назад»;
  dropdown — выпадающий список;
  edit - редактировать;
  reply - ответить;

**Текст:
  title, subject, heading, headline, caption — заголовок;
  subtitle — подзаголовок;
  lead, tagline — лид-абзац в тексте;
  text — текстовый контент;
  description, (desc) — описание, вариант текстового контента;
  excerpt — отрывок текста, обычно используется перед ссылкой «Читать далее…»;
  link, (lnk) — ссылка;
  copyright, copy — копирайт;

**Списки:
  list, items — список;
  item — элемент списка;

**Изображения:
  image, (img), picture, (pic) — картинка;
  icon, (ico) — иконка;
  pictogram - пиктограммма, знак, значёк;
  logotype, (logo) — логотип;
  thumbnail, (thumb) — миниатюра, уменьшенное изображение;
  background, (bg) - фоновое изображение;
  poster - плакат;

**Медиавыражения, устройства:
  phone, mobile, (mob) — мобильные устройства;
  phablet — телефоны с большим экраном (6-7″);
  tablet — планшеты;
  notebook, laptop — ноутбуки;
  desktop — настольные компьютеры;

**Размеры:
  tiny (xs - extra small) — маленький, крохотный;
  small, (sm) — небольшой;
  medium, (md) — средний;
  big, large, (lg) — большой;
  huge (xl - extra large) — огромный;
  narrow — узкий;
  wide, (wd) — широкий;

**Состояния:
  active, current — активный элемент, например, текущий пункт меню;
  hidden, invisible — скрытый элемент;
  visible - видимый;
  error — статус ошибки;
  warning — статус предупреждения;
  success — статус успешного выполнения задачи;
  default - значение по умолчанию;
  pending — состояние ожидания, например, перед сменой статуса на error или success;

**Адресные, географические данные:
  zip-code, (zip) - почтовый индекс;
  address - адрес;
  city - город;
  country - страна;
  build - строение, здание;
  auditory - помещение;
  
**Данные контрагента, партнера:
  legal-name - полное наименование организации;
  short-name - сокращенное наименование организации;
  TIN (Taxpayer Identification Number) - ИНН (Идентификационный Номер Налогоплательщика);
  IEC - КПП (Классификатор Промышленных Предприятий);
  legal-address - юридический адрес;

**Продуктовые данные:
  service - сервис, услуга, служба;
  product, prod - товар;
  product-code, vendor-code - артикул, номенклатурный номер;
  product-name - наименование товара;
  product-type - тип товара;
  product-category - категория товара;
  product-model, (model) - модель продукта;
  count - количество;
  price - цена;
  summ - сумма;
  total-sum, total-price - итого;
  sale - распродажа, скидка;
  order - заказ;
  purchase - покупка;
  category - категория;
  rate, rating - рейтинг;
  brand - фирма, марка;
  manufacturer, maker, producer - производитель;
  vendor - поставщик;
  specs - характеристики;
  cart, basket — корзина;

**Дополнительно:
  half - половина, полу-;
  part - часть;
  set - набор;
  property - свойство;
  post- (post-footer) - после-;
  pre- (pre-footer) - перед- , пред- ;
  bar - панель;
  search — поиск;
  advertisement, adv, commercial, promo — рекламный блок (режутся Адблоком, не рекомендуется использовать такие классы для блоков с внутренней рекламой);
  features, benefits — список основных особенностей товара, услуги
  meta — блок с дополнительной информацией, например, блок тегов и даты в посте;
  more, all — ссылка на полную информацию;
  short - сокращенное имя/наименование;
  full - полное;
  hack - костыль, хак;
  event - событие, мероприятие;
  task - задача;
  question, (quest) - вопрос;
  answer, (ans) - ответ;
  item - пункт;
  piece - кусок, участок, деталь, часть;
  unit - элемент, звено, составная часть;

-------------------------------------------------------------------*/
